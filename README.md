# Getting Started Lessons — Web Development Essentials

A compact, instructor-focused curriculum introducing core web development concepts: programming fundamentals, GitHub workflows, and accessibility best practices.

## Overview

This repository contains a short, beginner-friendly module set designed to onboard learners to professional web development tools and practices. Each module includes lesson content, examples, and assignments.

## How to use this repository

- To read lessons locally, open the files in a Markdown editor or on GitHub. Example: view `1-getting-started-lessons/README.md` for a module index.
- To customize or teach from this material, copy the module folder(s) into your course repository and adapt examples or assessments as needed.

Recommended local workflow:

```bash

code .
# Preview markdown in the editor or serve with a static site tool
# Example: use a Markdown preview extension or a simple static server
```

## Prerequisites

- A text editor (e.g., Visual Studio Code)
- Basic terminal/command-line familiarity
- Git (for Module 2 exercises)
